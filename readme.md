# One Trip

## Technologies Used

### Backend
- Django
- Django Rest Framework (REST API)
- Django Channels (WebSocket interface)
- SQLite (Only because I am not deploying this to the Play Store)
- Redis  (WebSocket cache)

### Deployment
- Daphne (ASGI Application server)
- Nginx
- Ubuntu

### Frontend
- Flutter
